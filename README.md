# 插件使用说明
# 插件作者：懒大猫

## 配置注意：

请配置指令前缀，需要和你配置文件里的一致，配置成其他不会有效果
# 重要注意：
## 如果你重启AstrBot，请在启动AstrBot完成后再次重载一次插件
以防比本插件落后加载的插件的指令没有被扫描到

# 跳过处理
### 指的是如果本插件符合该条件的话，就不处理这条消息，按原本该怎样就怎样
####
### 如果概率跳过设成0，则所有规则都没有通过的话，这条消息就不会再传下去，直接终止
### 如果概率跳过设成1，则所有规则都没有通过的话，这条消息直接传下去，相当于你只用了唤醒规则和指令规则
## 禁用指令与启用指令：

禁用的指令的优先级大于启用的指令，即优先判断是否为禁用的指令

### 额外指令：
针对可能没有扫描到的指令
### 指令前缀：
因扫描可能有问题，所有改为手动输入，反正一下子的事

如果你在多个规则里配置了同一个群聊，那么只会响应第一个规则的，不过一般人不会这么做
